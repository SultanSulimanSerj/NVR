# HTTP: Crow и потоки

Параметр `http.http_worker_threads` в `nvr.yaml` задаёт число рабочих потоков Crow (см. `HttpServer.cpp`). Значение `0` оставляет поведение библиотеки по умолчанию; для продакшена рекомендуется **≥ 2** при отдельном потоке на запись сегментов и тяжёлый архив.

Правило: не выполнять в обработчиках HTTP длительные блокирующие операции (экспорт архива уже вынесен в отдельные проверки размера/таймаутов).
