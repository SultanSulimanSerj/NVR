# Публичный SDK и OpenAPI

## Источник правды

Контракт HTTP описан в [openapi.yaml](openapi.yaml). Версия продукта API согласуется с полем `version` в `/api/v1/system/info`.

## Semver

- **MAJOR**: несовместимые изменения путей или семантики обязательных полей.
- **MINOR**: новые опциональные поля, новые эндпоинты.
- **PATCH**: исправления описаний, обратимые правки валидации.

Клиенты генерировать из OpenAPI 3 (например `openapi-generator-cli` для TypeScript/C#).

## Примеры

После стабилизации Tier 1 вынести в отдельный репозиторий `nvr-sdk-examples` (минимум: логин, список камер, таймлайн архива).
