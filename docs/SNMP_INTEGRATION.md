# SNMP (план)

Полноценные SNMPv2c/v3 trap **не реализованы** в текущем монолите. Операционная доставка событий для SIEM покрывается:

- **Syslog** — канал уведомлений `syslog` (UDP, RFC5424 JSON-тело события);
- **Webhook** с опциональным `webhook_hmac_sha256_key_base64` (заголовок `X-NVR-Signature: sha256=…`).

Для SNMP v2c trap к NMS типичен отдельный агент (`snmptrap`) или вынесенный коннектор; при появлении требований заказчика зафиксировать OID enterprise и формат varbinds в этом документе.
