# Платформа Tier 2: центральный сервер, облако, HA

Черновик границ для программы «Trassir-класс» вне одного edge-бинарника.

## Сервисы (MVP обсуждения)

| Сервис | Ответственность | Связь с edge |
|--------|-----------------|--------------|
| License / entitlements | выдача и отзыв лицензий, учёт каналов | короткоживущие токены capability |
| Config / fleet | шаблоны камер, политики, OTA | pull + подпись |
| Remote access | туннель (WireGuard / mTLS reverse), без прямого SSH на камеру | агент на edge или reverse-proxy |

## Угрозы (краткий threat model)

- Компрометация fleet API → массовая смена политик: **mTLS**, подпись тел конфигов, rate limit.
- Утечка лицензионного ключа: **привязка к fingerprint**, офлайн-кворум отзыва.
- Split-brain HA: явная политика кворума метаданных vs запись на shared storage.

## MVP scope (12–18 мес. ориентир)

Документировать API федерации edge↔core, без обязательства первой реализации в этом репозитории.
